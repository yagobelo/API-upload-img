## :sassy_man: Endpoints

- GET /picture - Lista todas as imagens
- GET /picture/:id - Busca uma imagens pelo id
- POST /picture - realiza o upload da imagem
- DELETE /picture/:id - Deleta uma imagem passando o id como parametro
